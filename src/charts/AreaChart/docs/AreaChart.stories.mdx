## Interactivity

The **area chart** provides next interactivity abilities:

| Action                                                      | Chart reaction                                                         |
| ----------------------------------------------------------- | ---------------------------------------------------------------------- |
| Hovering on the chart at some x-axis point                  | Tooltip for all areas at the current x-axis point is shown.            |
| Mouse wheel (or similar action on a laptop touchpad)        | Chart zooms in/out.                                                    |
| Dragging                                                    | Viewport is changing position relative to the chart.                   |
| Double click                                                | Chart zoom and viewport position return to the initial state.          |

### data

`type: object[]`

Specify the chart dataset via the **data** prop. <br />
The **data** prop includes information about areas positioning, names and colors. <br />
**This prop is required!**

A list of properties of the **data** prop object:

| Property name  | Type               | Default value  | Description                                                                                                                                                     |
| -------------- | ------------------ | -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **name**       | `string`           | **isRequired** | Name of the current area, that is displayed in the legend and tooltip.                                                                                          |
| **x**          | `number or string` | **isRequired** | Point position on **x**-axis.                                                                                                                                   |
| **y**          | `number or string` | **isRequired** | Point position on **y**-axis.                                                                                                                                   |
| **marker**     | `string`           | `none`         | Shape of the points in the area. May be one of the [supported symbols](#symbols).                                                                               |
| **fillColor**  | `string`           | `none`         | Color of the current area. If it is not specified, the area will be colored in one of default colors.                                                           |
| **lineColor**  | `string`           | `none`         | Color of the line, that is placed above the area.                                                                                                               |

### legendConfig

`type: object`

Custom config for rendering the legend. By default, the legend is visible and calculated from the [data](#data) prop. <br />
The **legendConfig** object must have next shape:

| Property name  | Type               | Default value  | Description                                                                                                                                                     |
| -------------- | ------------------ | -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **isVisible**  | `boolean`          | `true`         | Indicates whether the legend should be visible or not.                                                                                                          |
| **orientation**| `string`           | `vertical`     | Position of the legend regarding the chart. Must be one of: `'vertical'`, `'horizontal'`, `'horizontal-left'`.                                                  |
| **items**      | `object[]`         | `[]`          | Custom legend items.                                                                                                                                            |
| **hideAndKeepPlace**      | `boolean`         | `false`          | Indicates that legend should be invisible but keep its place.                                                                                                                                            |

The **items** property from the **legendConfig** prop must be of next shape:

| Property name       | Type     | Default value | Description                                                                                                                                                                             |
| ------------------- | -------- | ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **label**           | `string` | `''`          | Text, that is displayed in the legend for the current legend item.                                                                                                                      |
| **color**           | `string` | `none`        | Color of the symbol in the legend item.                                                                                                                                                 |
| **type**            | `string` | `none`        | Shape of the symbol. May be one of the [supported symbols](#symbols).                                                                                                                   |

#### Axis config

| Property name        | Type                                                    | Default value  | Description                                                                                                                                                                                                                                                                                                                                  |
| -------------------- | ------------------------------------------------------- | -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **label**            | `string`                                                | `''`           | Title of the axis, that is displayed below (x-axis) or to the left of axis (y-axis).                                                                                                                                                                                                                                                         |
| **domain**           | `[number, number]`                                      | `[]`           | Min and max initial values of the current axis. By default, it is calculated automatically from the [data](#data) prop.                                                                                                                                                                                                                      |
| **displayLine**      | `boolean`                                               | `true`         | Indicates whether the axis line should be visible or not.                                                                                                                                                                                                                                                                                    |
| **displayZeroLine**  | `boolean`                                               | `true`         | Indicates whether the axis zero line (line at 0 position) should be visible or not.                                                                                                                                                                                                                                                          |
| **displayGridLines** | `boolean`                                               | `true`         | Indicates whether the axis grid lines should be visible or not.                                                                                                                                                                                                                                                                              |
| **displayTickLabels**| `boolean`                                               | `true`         | Indicates whether the axis tick labels should be visible or not.                                                                                                                                                                                                                                                                             |
| **displayTicks**     | `boolean`                                               | `true`         | Indicates whether the axis tick lines should be visible or not.                                                                                                                                                                                                                                                                              |
| **displayLabel**     | `boolean`                                               | `true`         | Indicates whether the axis label (title) should be visible or not.                                                                                                                                                                                                                                                                           |
| **isZoomable**       | `boolean`                                               | `true`         | Indicates whether the ability to zoom in the current axis should be enabled or not.                                                                                                                                                                                                                                                          |
| **format**           | `object` of the [axis format shape](#axis-format-shape) | `{}`           | Formatting config for the current axis tick labels. You should use the [axis format shape](#axis-format-shape) for this property.                                                                                                                                                                                                            |
| **tickStep**         | `number`                                                | `0`            | For numeric axis it represents the step between axis tick values. For axis with string values it shows which labels should be shown with initial zoom. For example, with the **tickStep** `1` all tick values will be shown, with **tickStep** `2` each second will be shown. If `0` is passed, tick step will be calculated automatically.  |

#### Axis format shape

Values of axis tick labels can be formatted using this config:

| Property name         | Type                                                  | Default value  | Description                                                                                                                                               |
| --------------------- | ------------------------------------------------------| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **decimalPlaces**     | `number`                                              | `none`         | The length of the decimal part of the value.                                                                                                              |
| **prefix**            | `string`                                              | `''`           | A string that precedes the value.                                                                                                                         |
| **postfix**           | `string`                                              | `''`           | A string that is after the value.                                                                                                                         |

___

### decimalSeparator

`type: string`

A one-symbol string, that separates integer and decimal parts of values in the chart. By default, it is `'.'`.

### thousandSeparator

`type: string`

A one-symbol string, that separates every 3 symbols (thousands) of values in the chart. By default, there is no thousand separator.

### onAreaClick

`type: function`

Callback, that is called when clicking on any area. It receives the object with the next shape in params: `{ selectedAreas: object[] // A list of selected areas data }` <br />
For correct work of this prop the [enableAreasSelection](#enableareasselection) prop should be set to `true`.

An example of the **area chart** with specified `onAreaClick` callback:

<Preview>
  <Story name="With onAreaClick callback">
    <div style={{ height: 500, width: '100%' }}>
      <AreaChart
        data={[
          {
            name: 'State 1',
            x: ['2012-01', '2012-02', '2012-03'],
            y: [4000, 8000, 5000],
          },
          {
            name: 'State 2',
            x: ['2012-01', '2012-02', '2012-03'],
            y: [3000, 2000, 1000],
          },
        ]}
        isStacked={true}
        normalizedGroupType="percent"
        enableAreasSelection={true}
        onAreaClick={(selectedAreas) => {
          document.querySelector('#selected-areas').textContent = JSON.stringify(selectedAreas);
        }}
      />
      <div id="selected-areas">Selected areas</div>
    </div>
  </Story>
</Preview>
